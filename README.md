## Configuração

Clone o projeto

```sh
$ git clone https://github.com/pedrohcrisanto/pitzi_app.git
```

Instale as dependências
```sh
$ cd /pitzi_app/
$ bundle install
```

Crie o banco
```sh
$ rails db:create
```

Rode a aplicação
```sh
$ rails s
```

Abra o seu navegador(de preferência Chrome ou Firefox) e navegue para `localhost:3000/`
